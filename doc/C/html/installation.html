Getting the library

The Libgda library is bundled with some Linux distributions, so for these distributions, installing can be done using the distribution specific tools.

Also Libgda's sources are available for download from the GNOME ftp servers, and from the GNOME SVN repositories; see the Gnome-DB web site for more information.
