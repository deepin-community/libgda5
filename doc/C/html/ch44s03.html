Methods - provider's information

get_name() - mandatory

This method returns the name of the provider; it should only contain alphanumeric characters.

get_version() - mandatory

This method returns the version of the provider (providers built as part of Libgda have the same version as Libgda).

get_server_version() - mandatory

This method returns the version of the database server to which a connection is opened

supports_feature()

This method allows one to ask the provider if some specific features are available
