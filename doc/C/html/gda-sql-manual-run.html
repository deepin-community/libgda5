Features

The Libgda's console tool' main features are:

  • Handle more than one connection at the same time

  • Report meta data in a similar way, whatever the real type of database accessed

  • Create virtual connections binding several already opened connections (to run SQL commands across multiple connections)

  • Allow one to define variables which can be referenced in any SQL statement (using a common syntax)

  • One query buffer per connection to edit complex queries (uses a parameter-able external editor)

  • Easy command line editing and output using the Readline library and a (parameter-able) pager

  • Work on any platform on which Libgda has been ported to.

  • List, define or remove named data sources (DSN)

  • Import and export BLOBS (binary large objects) from and to local files

  • On command embedded HTTP server
